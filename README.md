# How to start server

```bash

# init package.json file
$ npm init -y

# install package
## dependencies: những thư viện phục vụ cho sử dụng web -> npm install xxx
## devDependencies: những thư viện phục vụ cho việc dev -> npm install --save-dev xxx
$ npm install xxx

# package first install
$ npm i express (https://www.npmjs.com/package/express)

# run entry file
$ node index.js

# package nodemon install
$ npm install --save-dev nodemon (https://www.npmjs.com/package/nodemon)


```
